My thoughts on ReactJS

Talking about my thoughts and interests on using ReactJS to build UI components.

Abdush Shakoor ● March 13th, 2020

Over the past few months, I have been building some test projects using ReactJS to understand the concepts and fundamentals of it and I've got to admit, I really like it.

Here are some of the things that I have built:

  • Ideaboard
  • Restaurant Search
  • Time Tracker

What is ReactJS?

Initially, I was focused on building components using vanilla JavaScript and jQuery because I got frustrated with ReactJS's fundamentals and concepts. But once I started to learn it, I realized why it might be worth learning it.

ReactJS is a JS library written by Facebook's frontend engineering team. It's an open-source project with over 100K stars on GitHub. It's created with the aim of creating reusable UI components, you can think ReactJS to be the View of an Model-View-Controller application.

Some of the things I found that are interesting is how ReactJS streamlined the process of handling data and events by using state and props.

This ain't a tutorial but if you want to know more about ReactJS, read their documentation on their official website.

In this article, I'll talk about some of the topics that were of interest to me:

  • Thinking in React
  • Managing events
  • State vs Props
  • Conditional Rendering
  • Custom hooks
