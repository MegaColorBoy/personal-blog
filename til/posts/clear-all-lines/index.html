VIM

Clear all lines

Switch to normal mode by pressing keyPress and it will take you to first line of the file.Then type and it will clear all the lines from start to the end of the file.

1 minute read

  1. Switch to normal mode by pressing ESC key
  2. Press gg and it will take you to first line of the file.
  3. Then type dG and it will clear all the lines from start to the end of the file.
